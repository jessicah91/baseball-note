배포 방법

현재 구조
- 메인 온보딩: 닉네임 + 응원 구단 선택
- 허브 페이지: 덕아웃노트 / 구단순위 / 경기룰 / 마이페이지
- 덕아웃 노트 저장 후 마이페이지에서 모아보기 가능
- 전 페이지 공통 나이트/데이 모드 가능

참고
- 구단순위 페이지는 현재 연동 자리만 잡아둔 상태
- 나중에 KBO 공식 연동을 붙일 수 있게 페이지 구조를 분리해둠
